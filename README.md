# CoilCalc - Vape Coil Rechner

Eine Android-App zur Berechnung von Vape Coil-Parametern.

## Beschreibung

CoilCalc ist eine kostenlose Android-Anwendung, die auf dem Google Play Store verfügbar ist. Die App hilft Dampfern dabei, wichtige Parameter für ihre Coil-Wicklungen zu berechnen.

## Features

- Berechnung von Coil-Parametern
- Benutzerfreundliche Oberfläche
- Mehrsprachige Unterstützung (Deutsch, Französisch)

## Installation

1. Klonen Sie das Repository:
```bash
git clone https://github.com/vape-laden/CoilCalc.git
```

2. Öffnen Sie das Projekt in Android Studio

3. Bauen und starten Sie die App auf einem Android-Gerät oder Emulator

## Entwicklung

### Voraussetzungen

- Android Studio
- Android SDK
- Gradle

### Build

```bash
./gradlew build
```

## Lizenz

Diese App ist auf dem Google Play Store öffentlich und kostenlos verfügbar.
